# fitodizain
Фитодизайн Интерьера

Описание

Мобильное приложение для фитодизайна интерьера, которое помогает пользователям выбирать растения для оформления пространства, предоставляет рекомендации по уходу и позволяет создавать дизайн-проект.

Основные функции

Каталог растений: обширная база данных с информацией о различных видах растений.

Рекомендации по уходу: советы по поливу, освещению и другим условиям для каждого растения.

Дизайн-проект: инструмент для создания и сохранения проектов оформления интерьера.

Уведомления: напоминания о поливе и других важных событиях.

Установка

Скачайте APK-файл из репозитория.

Установите приложение на ваше Android-устройство.

Использование

Запустите приложение.
Выберите растение из каталога или воспользуйтесь поиском.
Создайте дизайн-проект, добавляя растения в виртуальное пространство.
Получайте уведомления и рекомендации по уходу.
Технические требования
Android 5.0 и выше.
Минимальное разрешение экрана: 720p.
